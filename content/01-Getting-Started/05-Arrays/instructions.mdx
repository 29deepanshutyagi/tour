# Arrays

We will convert the hobbies field to an array of strings. 
  
```json highlightLineStart={4}
{
  "name": "John Doe",
  "age": 25,
  "hobbies": ["reading", "writing"]
}
```


In _JSON Schema_, defining an array is similar to defining an object. Use the `type` keyword with the value `array` to define an array. now use the `items` keyword to define the type (or _subschema_) of items in the array. 

**Example:**
```json
{
  "skills": {
    "type": "array",
    "items": {
      "type": "string"
    }
  }
}
```


Now, try to modify the `hobbies` property on the right side editor and convert it to an **array of strings**.
