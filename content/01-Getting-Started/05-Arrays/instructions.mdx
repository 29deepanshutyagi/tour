# Arrays

Let's convert the hobbies field to an array of strings. 
  
```json highlightLineStart={4}
{
  "name": "John Doe",
  "age": 25,
  "hobbies": ["reading", "writing"]
}
```


In _JSON Schema_, defining an array is similar to defining an object. Use the `type` keyword with the value `array` to define an array. 

You can use the `items` keyword to define the *schema* of the array elements. The value of items should be a [subschema](https://json-schema.org/learn/glossary#subschema).

**Example:**
```json
{
  "skills": {
    "type": "array",
    "items": {
      "type": "string"
    }
  }
}
```
