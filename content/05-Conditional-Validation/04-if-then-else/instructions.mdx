# If-Then-Else


 **if** a condition is true, **then** apply a [subschema](https://json-schema.org/learn/glossary#subschema). **else**, apply another [subschema](https://json-schema.org/learn/glossary#subschema).
  
Consider an `isFullTime` Field. **If** `isFullTime` is true, **then** the `salary` field must be present, **else** the `hourlyRate` field must be present.



```json highlightLineStart={3} highlightLineEnd={4}
{
  "name": "John Doe",
  "isFullTime": true,
  "salary": 50000
}
```
You can use the `if-then-else` [keyword](https://json-schema.org/learn/glossary#keyword) to enforce this condition.

**Example Schema**

```json highlightLineStart={8} highlightLineEnd={10}
{
  "type": "object",
  "properties": {
    "isFullTime": {"type": "boolean"},
    "salary": {"type": "number"},
    "hourlyRate": {"type": "number"}
  },
  "if": {"properties": {"isFullTime": {"const": true}}},
  "then": {"required": ["salary"]},
  "else": {"required": ["hourlyRate"] }
}
```
